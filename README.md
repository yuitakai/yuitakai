# Yui Takai personal homepage

A static researcher homepage with:

- Japanese / English / French language switching
- light-blue, water-inspired design
- thin readable typography using Inter and Noto Sans JP
- click ripple animation
- sections common on researcher websites: About, Research, Publications, CV, News, Contact

## How to edit

1. Open `index.html` and replace `your.email@example.com` with your email.
2. Replace the profile links (`Google Scholar`, `ORCID`, `GitHub`) with your actual URLs.
3. Replace the portrait placeholder with a photo if desired.
4. Edit text in `assets/script.js` to update Japanese, English, and French copy.

## Deploy options

### GitHub Pages

- Create a repository such as `yuitakai.github.io`.
- Upload these files to the repository root.
- In Settings → Pages, publish from the main branch.
- Keep the `CNAME` file if using `yuitakai.com`; change it if using another domain.

### Netlify

- Drag and drop this folder into Netlify, or connect it to a GitHub repository.
- Add your custom domain in the Netlify domain settings.
